# 🧠 Kanban Board (Full Stack)

Aplicação Kanban desenvolvida com **React, Node.js, Express e PostgreSQL**, permitindo gerenciar tarefas com drag and drop e persistência em banco de dados.

---

## 🚀 Demo

🔗 https://kanban-react-rouge.vercel.app/

---

## 🧱 Arquitetura

```
Frontend (React)
       ↓
Backend API (Node + Express)
       ↓
PostgreSQL (Neon)
```

---

## ⚙️ Tecnologias

### Frontend

* React
* Vite
* @dnd-kit (Drag and Drop)

### Backend

* Node.js
* Express

### Banco de Dados

* PostgreSQL (Neon)

### Deploy

* Vercel (Frontend)
* Render (Backend)

---

## 🔥 Funcionalidades

* ✅ Criar tarefas
* ✅ Deletar tarefas
* ✅ Atualizar status (drag and drop)
* ✅ Persistência em banco de dados
* ✅ API REST completa (CRUD)


## 🛠️ Como rodar localmente

### Backend

```bash
cd backend
npm install
node server.js
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
