# Kanban

Um Kanban básico desenvolvido utilizando React e Node.js + Express

Ele permite a criação de tarefas, manusear tais tarefas entre suas colunas e também deletar essas tarefas.

 Frontend

- React
- Vite
- dnd-kit
- CSS

 Backend

- Node.js
- Express
- Cors

 Funcionalidades
 Criar tarefas
- Deletar tarefas
- Arrastar tarefas entre colunas
- Atualizar status das tarefas
- API REST para gerenciamento de tarefas
